Tensors

What is a tensor?

  • Generalization of:
    • Scalar (rank 0)
    • Vector (rank 1)
    • Matrix (rank 2)
    • Higher dimensions (rank ≥ 3)

Creating Tensors

import numpy as np
import tensorflow as tf

a = tf.constant(3)
b = tf.constant([1, 2, 3])
c = tf.constant([[1., 2.], [3., 4.]])

Tensor Properties

c.shape
c.dtype
tf.rank(c)
<tf.Tensor: shape=(), dtype=int32, numpy=2>

Key Idea

Tensors have shape, dtype, and rank

Tensor Operations

TensorFlow feels like NumPy:

x = tf.random.normal((3, 2))
y = tf.ones((3, 2))

x + y
x * y

Matrix multiplication:

tf.matmul(x, tf.transpose(y))
<tf.Tensor: shape=(3, 3), dtype=float32, numpy=
array([[-0.44380572, -0.44380572, -0.44380572],

Important Tensor Rules

  • Tensors are immutable
  • Operations are GPU-aware
  • Broadcasting rules apply (like NumPy)

Functional API

inputs = tf.keras.Input(shape=(10,))
x = tf.keras.layers.Dense(32, activation="relu")(inputs)
outputs = tf.keras.layers.Dense(1)(x)

model = tf.keras.Model(inputs, outputs)

Use when:

  • Multiple inputs / outputs
  • Complex architectures

Training Workflow

The 3-Step Pattern

  • compile
  • fit
  • evaluate / predict

This pattern appears everywhere in TF2.

Compile Step

model.compile(
    optimizer="adam",
    loss="mse",
    metrics=["mae"]
)
  • Optimizer → how parameters change
  • Loss → what we minimize
  • Metrics → what we monitor

Fit the Model

X_train, y_train = np.random.rand(100, 10), np.random.rand(100, 1)  # Training data
history = model.fit(
    X_train,
    y_train,
    epochs=5,
    batch_size=32,
    validation_split=0.2
)
