Causal Self-Attention

Causal (Masked) Attention

Prevent future leakage:

\[ \text{Mask}_{i,j} = \begin{cases} 0 & j \le i \\ -\infty & j > i \end{cases} \]

Masked Attention Formula

\[ \text{softmax}\left(\frac{QK^T + M}{\sqrt{d_k}}\right) \]

Why Masking?

During training:

  • input: “The cat sat”
  • target: “The cat sat”

Must not see future tokens

Visual: Causal Mask

\[ \begin{bmatrix} 1 & 0 & 0 & 0 \\ 1 & 1 & 0 & 0 \\ 1 & 1 & 1 & 0 \\ 1 & 1 & 1 & 1 \end{bmatrix} \]

Positional Encoding

Positional Encoding (Problem)

Problem

Self-attention ignores order!

Solution: - Add positional information

Sinusoidal Encoding

\[ \text{PE}(p, 2i) = \sin\left(\frac{p}{K^{2i/d}}\right) \]

\[ \text{PE}(p, 2i+1) = \cos\left(\frac{p}{K^{2i/d}}\right) \]

Properties

  • Unique representation per position
  • Linear relation between positions
  • No learned parameters

Why Summation Works

\[ \mathbf x_{\text{final}} = \mathbf x_{\text{embedding}} + \text{PE} \]

Because: - same dimension - model learns to separate signals

Intuition

  • Embedding → what
  • Position → where

Masked Attention in TensorFlow 2

mask = tf.linalg.band_part(tf.ones((seq_len, seq_len)), -1, 0)

import tensorflow as tf

x = tf.random.uniform((1, 3, 4))  # (batch_size, seq_len, d_model)
seq_len = tf.shape(x)[1]

attention = tf.keras.layers.MultiHeadAttention(
    num_heads=4,
    key_dim=64
)

mask = tf.linalg.band_part(tf.ones((seq_len, seq_len)), -1, 0)

output = attention(
    query=x,
    value=x,
    key=x,
    attention_mask=mask
)

Decoder Block in TensorFlow 2

class DecoderBlock(tf.keras.layers.Layer):
    def __init__(self, d_model, num_heads):
        super().__init__()
        self.attn = tf.keras.layers.MultiHeadAttention(num_heads, d_model)
        self.ffn = tf.keras.Sequential([
            tf.keras.layers.Dense(4*d_model, activation='relu'),
            tf.keras.layers.Dense(d_model)
        ])
        self.norm1 = tf.keras.layers.LayerNormalization()
        self.norm2 = tf.keras.layers.LayerNormalization()

    def call(self, x, mask=None):
        attn_output = self.attn(x, x, attention_mask=mask)
        x = self.norm1(x + attn_output)
        ffn_output = self.ffn(x)
        return self.norm2(x + ffn_output)
